## What Changed

### Task 1: safeJsonParse helper and bare JSON.parse replacement
- Added `safeJsonParse<T>(raw, fallback)` to `utils/storage.ts` -- wraps JSON.parse in try/catch with typed fallback
- Replaced all 12 bare `JSON.parse` calls on SecureStore values across 10 files with `safeJsonParse`
- Wrapped 3 DB-column `JSON.parse(adjustments)` calls in try/catch with null fallback (map.tsx, room/[code].tsx, index.tsx)
- **Commit:** 1395353

### Task 2: Realtime null guards, map marker safety, unread cleanup, subscribe errors

- `handleLeaveRoom` and `handleLeaveMultipleRooms` now clean unread state from both in-memory Set and SecureStore
- Both `.subscribe()` calls now include `(status, err)` callback that logs errors
