1. In `utils/storage.ts`, add and export a `safeJsonParse` helper:
   ```ts
   export function safeJsonParse<T>(raw: string | null, fallback: T): T {
     if (!raw) return fallback;
     try { return JSON.parse(raw) as T; } catch { return fallback; }
   }
   ```

2. Replace every bare `JSON.parse` on SecureStore values across the codebase with `safeJsonParse(raw, fallback)`. The fallback should match the existing default used at each call site:

   - `utils/rooms.ts:21` — `safeJsonParse(raw, [])` (was `raw ? JSON.parse(raw) : []`)
   - `utils/lastSeen.ts:8` — `safeJsonParse(raw, {})` (both lines 8 and 15)
   - `utils/nicknames.ts:40` — `safeJsonParse(raw, {})`
   - `utils/avatar.ts:33` — `safeJsonParse(raw, null)`, then `if (parsed) return parsed; return DEFAULT_AVATAR;`
   - `utils/sunset.ts:113` — wrap in try/catch OR use safeJsonParse with null fallback, skip cache on null
   - `utils/messages.ts:23,54` — `safeJsonParse(raw, [])`
   - `app/(tabs)/chats.tsx` — every `JSON.parse(raw)` that reads from SecureStore (lines ~90, 99, 333, 685, 1115) use `safeJsonParse(raw, {})` or `safeJsonParse(raw, [])` as appropriate
   - `app/room/[code].tsx` — lines ~182, 267 use `safeJsonParse(raw, {})`
