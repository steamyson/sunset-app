<objective>
Guarantee overlap-free cloud layout on every canvas load and after every new room join, with position persistence and proportional shrink.

Purpose: SKY-01 and SKY-02 require that no two room clouds ever overlap on the sky canvas. The current `findNonOverlappingPosition` uses grid+jitter but never verifies the final layout is collision-free. This plan adds a shrink-until-fits loop, SecureStore position persistence (all-or-nothing per D-04), and a spring scale-in animation for newly added clouds.

Output: Modified `app/(tabs)/chats.tsx` with overlap-free layout guarantee, position persistence, and new-cloud animation.
